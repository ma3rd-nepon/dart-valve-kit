# Valve Kit

[![pub version](https://img.shields.io/pub/v/valve_kit)](https://pub.dev/packages/valve_kit)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Dart package for work with Valve formats (VDF, VPK, VTF, etc.).

---

## Features

| Format | Feature | Status |
|--------|---------|--------|
| **VDF** | read/write | *WIP*
| **VPK** | read | *WIP*
| **VTF** | read | *WIP*
| **BSP** | read | *WIP*

---

## Instalation

```bash
dart pub add valve_kit
```

or in pubspec.yaml

```yaml
dependencies:
  dart_valve_kit: any
```
